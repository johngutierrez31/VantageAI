# Vantage Demo Walkthrough Planner

## Trigger conditions

- Demo walkthrough creation
- Demo click-path planning
- Wow-path refinement
- Leadership or buyer-facing product demo prep

## Required behavior

1. Start from real seeded records and current in-app routes only.
2. Prefer `Command Center` and `Tools Hub` as the entry points.
3. Build both a short path and a fuller module tour.
4. Keep the story coherent: buyer diligence, executive visibility, governed AI, and live response carry-over.
5. Name the artifact the viewer ends with at each major step.
6. Do not reference unsupported exports, missing pages, or unconfirmed modules.

## Structured output

- `best_click_path`
- `three_minute_version`
- `ten_minute_version`
- `wow_moment_sequence`
- `artifact_highlights`
- `talking_points`

## Product mapping

- Route viewers into seeded TrustOps, Pulse, AI Governance, and Response Ops records.
- Keep walkthrough guidance aligned with current help docs, Command Center, and Tools Hub launchers.
- Treat Copilot as a workflow launcher, not the main demo destination.

# Policies, Runbooks, Security Analyst, and Cyber Range

## Policies

Use Policies to generate and manage policy material that supports trust, governance, and operational readiness.

Best use cases:

- core security policies
- customer trust materials
- AI governance policy support
- incident response policy support

## Runbooks

Use Runbooks to create repeatable action packs for operational work.

Best use cases:

- recurring security processes
- response task generation
- standardizing manual procedures

## Security Analyst

Use Security Analyst when you want guided analysis of security posture, priorities, or concerns.

Best use cases:

- quick analysis before leadership conversations
- structured operator reviews
- turning concerns into next-step recommendations

## Cyber Range

Use Cyber Range to generate or refine realistic security exercise plans.

Best use cases:

- exercise planning
- scenario design
- testing security assumptions
- preparation for tabletop or response-readiness reviews

# Response Ops Overview

## What Response Ops is for

Response Ops is the incident and exercise layer of VantageAI.

Use it when the organization needs a durable incident record, fast first-hour guidance, runbook task packs, timeline capture, and a formal after-action closeout.

## Response Ops pages

- Response Ops dashboard
- Incident detail pages
- After-action reporting
- Tabletop exercise pages
- linked runbook and finding surfaces

## Use it when

- an event needs formal triage
- an incident needs owned task execution
- you need a decision trail during a live issue
- a post-incident review needs to create real follow-up work
- you want lightweight tabletop exercises without a separate platform

## Core workflow

1. Start incident triage
2. Launch the runbook pack
3. Assign first-hour actions
4. Capture timeline events and decisions
5. Generate the after-action report
6. Create follow-up findings, tasks, or risks
7. Feed the outputs back into Pulse and quarterly review workflows

## Why it matters

Response Ops makes the suite credible during live operational pressure, not just during governance and trust cycles.

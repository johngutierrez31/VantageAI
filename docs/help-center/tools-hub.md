# Tools Hub

## What this page is for

Tools Hub is the guided launcher for the suite.

It is for users who know the outcome they want but do not want to navigate the product by module.

## When to use it

Use Tools Hub when you want to launch a workflow such as:

- Draft Questionnaire Answers
- Build Evidence Map
- Promote Approved Answers to Library
- Review TrustOps Queue
- Assemble Buyer Trust Packet
- Generate Executive Scorecard
- Build Risk Register
- Generate 30/60/90 Roadmap
- Draft Board Brief
- Prepare Quarterly Review
- Register AI Use Case
- Start AI Vendor Intake
- Start Incident Triage
- Launch Runbook Pack
- Draft After-Action Report
- Prepare Tabletop

## How to use it

1. Start with the problem you need to solve
2. Pick the workflow that best matches the output you need
3. Read what the launcher says it will produce
4. Run the workflow
5. Move to the created record for review, approval, or export

## Best use cases

- first-time user onboarding
- fast launch for recurring security work
- minimizing training overhead for part-time reviewers or operators

## Tips

- Use Tools Hub when you want action, not exploration
- Use Command Center when you want prioritization
- Use Copilot when you want guided help choosing the right workflow
